description: Create or update the memory index for quick navigation
---

Update the memory index to help me navigate all my daily entries:

1. **Scan all memory folders** - Look in `memory/` for all `YYYY-MM-DD` folders
2. **Extract metadata** from each day:
   - Date
   - Main topics from notes.md (first line/summary)
   - Key accomplishments
   - Status/tags (Active, Completed, etc.)

3. **Update memory/INDEX.md** with:
   - A table organized by month/year with dates, summaries, and links
   - Count of total entries
   - Recent entries highlighted
   - Quick navigation by month

4. **Add helpful sections**:
   - "Recent Memories" - Last 5 days
   - "Completed Projects" - Mark entries with final status
   - "Quick Links" - Most referenced or important entries

The index should make it easy to find past work and track progress over time.
